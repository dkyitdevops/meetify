# AGENTS.md

## Каждая сессия
При старте запусти memory-navigator — прочитай INDEX.md и свежий лог.
Проверь есть ли незавершённые задачи из прошлой сессии.

## Память
- memory-navigator — ориентация при старте и поиск по прошлому
- memory-writer — «запомни» / «запиши» → сохранить факт навсегда
- memory-gardener — раз в неделю, по запросу «разбери память»

Правило: MEMORY.md не более 8KB (индекс). Детали — в domain-файлах.
[needs-verify] на всё что не сказал пользователь лично.

## Анализ сессии
Слово «анализ» → просмотри диалог → запиши важное через memory-writer.
Бэкап выполняется автоматически каждую ночь — вручную не нужно.

## Безопасность
Токены и ключи → credentials/{service}.env, chmod 600, записать в memory/secrets.md.
Не передавай данные третьим сервисам без разрешения.

## Форматирование (Telegram)
Без таблиц → списки. Без заголовков в коротких ответах.
