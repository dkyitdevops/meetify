## Модуль 1: Работа с кириллицей в JavaScript (Frontend-агенты)

### Проблема
Агенты не справились с несоответствием ключей (`id` vs `name`, регистр букв) для агентов с кириллическими именами.

### Что изучить

1. **Unicode и нормализация строк**
   - `String.prototype.normalize()`
   - Разница между NFC, NFD, NFKC, NFKD
   - Проблемы с регистром в кириллице

2. **Консистентность ключей в объектах**
   ```javascript
   // Антипаттерн (что делали агенты):
   const key1 = agent.id;      // "елена"
   const key2 = agent.name;    // "Елена"
   const key3 = agent.agent;   // "Елена"
   
   // Правильный подход:
   const normalizedKey = (agent.name || agent.agent || agent.id).toLowerCase().trim();
   ```

3. **Отладка проблем с данными**
   - `console.table()` для объектов
   - `Object.keys()` проверка
   - Сравнение MD5/хэшей данных

4. **Практическое задание**
   - Создать систему с агентами "Алексей", "Елена", "Мария"
   - Обеспечить консистентность ключей
   - Написать тесты

## Модуль 3: Деплой и DevOps (DevOps-агенты)

### Проблема
Файлы не всегда попадали в правильное место на сервере, MD5 суммы не совпадали.

### Что изучить

1. **Контроль целостности деплоя**
   - MD5/SHA256 проверка файлов
   - Сравнение локального и remote состояния
   - Rollback стратегии

2. **Docker и контейнеры**
   - Bind mounts vs volumes
   - Кэширование слоёв
   - Правильное копирование файлов

3. **Автоматизация деплоя**
   - GitHub Actions
   - Проверка после деплоя
   - Health checks

4. **Практическое задание**
   - Настроить CI/CD pipeline
   - Добавить проверку MD5 после деплоя
