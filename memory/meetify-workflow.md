# Рабочий процесс разработки Meetify

## Обязательные шаги после каждой новой фичи

### 1. Автотесты
- [ ] Regression tests: `node regression.test.js`
- [ ] Multi-user tests: `node multi-user.test.js`
- [ ] Whiteboard tests: `node whiteboard.test.js`
- [ ] UI tests: запуск через GitHub Actions (ui-tests-v2.yml)
- [ ] При необходимости — корректировать тесты

### 2. Анализ кода
- [ ] Синтаксис: `node --check file.js`
- [ ] HTML валидность
- [ ] Уязвимости (CodeQL в GitHub Actions)

### 3. Отчет пользователю
- [ ] Результаты всех тестов
- [ ] Статус анализа кода
- [ ] Список изменений
- [ ] Процент готовности
