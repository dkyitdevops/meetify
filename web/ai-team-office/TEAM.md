# AI Team Office — Команда агентов

## Проблема
Агенты в режиме `mode="run"` завершаются после выполнения задачи и становятся недоступны для `sessions_send`.

## Решение
Создавать агентов с длинным таймаутом и проверять статус перед отправкой задачи.

## Текущая команда (актуальная)

| Роль | Статус | Последняя задача |
|------|--------|------------------|
| 🎨 UI Designer | ✅ Done | Проект на мониторах |
| 💻 Frontend | 🔄 Running | Исправление стилей |
| ⚙️ Fullstack | ✅ Done | Безопасность |
| 🎬 Animator | ✅ Done | Анимации |
| 🧪 QA Engineer | ✅ Done | Тестирование |
| 🚀 DevOps | ✅ Done | CI/CD |
| 🔒 Security | ✅ Done | Аудит |
| 📚 Аналитик | ✅ Done | Документация |
| 📋 КЭП ТСО | ✅ Ready | Консультации |

## Процесс работы

1. Проверить статус агента через `subagents list`
2. Если `done` — создать нового агента с той же ролью
3. Отправить задачу новому агенту
4. Агент выполняет и завершается
5. Повторить при необходимости
