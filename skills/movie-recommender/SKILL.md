# Movie Recommender

## Когда активируется этот скилл
- "что посмотреть", "что бы посмотреть", "what to watch"
- "посоветуй фильм / сериал"
- "добавь в вишлист", "хочу посмотреть X"
- "посмотрел X", "досмотрел X", "закончил X"
- "смотрю сейчас X", "начал смотреть X"
- Пользователь делится эмоциями: "X был потрясающим", "X разочаровал", "X — шедевр"
- Пользователь обсуждает фильм: сюжет, актёров, режиссёра, сравнения
- Пользователь присылает список фильмов с оценками для импорта

## Файлы
- `memory/watchlist.md` — очередь и просмотренные
- `memory/kinopoisk_ratings.md` — история оценок
- `memory/movie-profile.md` — кешированный профиль вкуса (создать если нет)

Формат записей и алгоритм совместимости — см. `reference.md`.

---

## Режим 1: "Что посмотреть"

1. Прочитать `memory/watchlist.md`, показать 2-3 фильма со статусом 🟡, сортировка по прогнозу совместимости (высокий → низкий)
2. Если 🟡 меньше 2 или пользователь отказался — перейти к **Поиску по профилю**

### Поиск по профилю (когда нужно что-то новое)

**Важно: сначала профиль, потом поиск**

1. Прочитать `memory/movie-profile.md` (если есть) — использовать готовый профиль
2. Если профиля нет — прочитать `memory/kinopoisk_ratings.md`, вывести паттерны (топ-жанры, топ-режиссёры, топ-темы по оценкам 9-10)
3. Перевести запрос пользователя через призму профиля (см. таблицу в reference.md)
4. Сделать веб-поиск с персонализированным запросом
5. Проверить результаты против `kinopoisk_ratings.md` — исключить уже просмотренные
6. Объяснить связь: "Ищу X, потому что у тебя [фильм] — 10/10"

---

## Режим 2: Добавление фильма в вишлист

1. Разобрать: название, год, источник (если упомянут)
2. Проверить дубли в `watchlist.md` и `kinopoisk_ratings.md`
3. Рассчитать прогноз совместимости (см. reference.md)
4. Добавить запись по шаблону из reference.md
5. Обновить таблицу статистики в начале `watchlist.md`

---

## Режим 3: После просмотра

1. В `watchlist.md`: статус 🟡 → 🟢, перенести в секцию "Просмотрено"
2. Спросить оценку (1-10) и впечатления
3. В `kinopoisk_ratings.md`: добавить/обновить запись с оценкой и заметками
4. Обновить статистику в `watchlist.md`
5. Если профиль `movie-profile.md` существует — предложить обновить его

---

## Режим 4: Обсуждение фильма / эмоции

Когда пользователь упоминает фильм в разговоре (не с целью добавить в список или спросить что смотреть):

1. Найти фильм в `kinopoisk_ratings.md` — знать текущую оценку и статус
2. Если фильм в `watchlist.md` со статусом 🟡 — не спойлерить
3. **Фиксировать комментарии:** если пользователь выражает конкретное мнение о фильме (не просто "хороший") — предложить сохранить как заметку:
   > "Записать это в заметки к фильму?"
4. Если пользователь говорит что смотрит сейчас — обновить статус на ⏳, не спойлерить
5. Если комментарий достаточно содержательный и пользователь согласен — добавить в поле `**Мои заметки:**` в `kinopoisk_ratings.md`

### Когда сохранять заметку без явного согласия

Сохранять автоматически (без переспрашивания) если пользователь:
- Явно просит: "запомни", "отметь", "зафиксируй"
- Даёт развёрнутое мнение из 2+ предложений о конкретном фильме

Переспрашивать если:
- Мнение мимолётное ("да, норм фильм")
- Непонятно, имеет ли пользователь в виду конкретный фильм или жанр в целом

---

## Режим 5: Массовый импорт оценок

Когда пользователь присылает список фильмов с оценками:

1. Определить формат (текст, CSV, JSON, построчно)
2. Для каждого фильма:
   - Проверить дубли в `kinopoisk_ratings.md`
   - Добавить в базовом формате (только оценка + жанры если есть)
   - Оценки 9-10: добавить в расширенном формате с описанием
3. Обновить статистику в шапке `kinopoisk_ratings.md`
4. Сообщить итог: добавлено N, пропущено M (дубли), требует уточнения K
5. Предложить обновить `movie-profile.md` после импорта

---

## Формат ответа

**Рекомендация из вишлиста:**
```
Из твоей очереди:

🎬 [Название] ([Год]) — [1-2 предложения]
Совместимость: [score]/10 — [краткое обоснование]
Источник: [кто/откуда]
```

**Новая рекомендация (поиск):**
```
На основе твоих оценок ([паттерн]):

🎬 [Название] ([Год]) — [описание]
Похоже на: [фильм из высоких оценок пользователя]
Совместимость: [score]/10

Добавить в очередь?
```

**Подтверждение сохранения заметки:**
```
Сохранил к "[Название]": [цитата или суть заметки]
```
