# Reference: Movie Recommender Formats & Algorithm

## Entry Template for watchlist.md

```markdown
### [Название на русском] / [Original Title] ([Год])
**Источник:** [кто/где рекомендовал]
**Статус:** 🟡 в очереди | ⏳ смотрю | 🟢 просмотрен | 🔴 отложен

**Жанры:** [жанры через запятую]
**Режиссёр:** [имя]
**В главных ролях:** [актеры, если важно]
**Краткое описание:** [2-3 предложения о сюжете]

**Прогноз совместимости:** [1-10]/10
**Обоснование:** [почему этот рейтинг — на основе оценок пользователя]

**Почему в очереди:** [мысли при добавлении]
**Теги:** #[источник] #[жанр] #[приоритет] #[дата_добавления]
```

## Compatibility Score — Formula

Базовый балл: 5 (нейтральный, если нет данных)

Корректировки (суммировать):

**+2** — жанр совпадает с топ-3 жанрами пользователя по оценкам 9-10
**+1** — жанр встречается в оценках 7-8
**-1** — жанр пользователь оценивает в среднем ниже 6
**+2** — режиссёр уже оценён пользователем на 9-10
**+1** — режиссёр оценён на 7-8
**+1** — тема/атмосфера совпадает с топ-темами (философия, ИИ, семья, детектив — смотреть по тегам)
**+1** — страна/регион с высоким средним (например, Корея если много корейских высоко)
**-1** — длинная франшиза, если пользователь редко ставит продолжениям высокие оценки
**+1** — рекомендовал знакомый (источник = имя человека)

Минимум: 1. Максимум: 10. Округлять до целого.

При < 10 рейтингов в базе: не ниже 5, не выше 8.

## Duplicate Detection

Check both files for:
- Exact title match (case-insensitive)
- Similar title with same year
- Original title match

If found → inform user and show existing entry.

## Status Transitions

```
🟡 в очереди → ⏳ смотрю (когда пользователь начал)
⏳ смотрю → 🟢 просмотрен (после завершения)
🟡 в очереди → 🟢 просмотрен (если не упоминал начало)
🟡 в очереди → 🔴 отложен (не интересно сейчас)
🔴 отложен → 🟡 в очереди (передумал)
```

## Web Search Trigger

Use web search when:
- Watchlist has < 2 items with 🟡 status
- User explicitly asks for "something new" / "не из вишлиста"
- User requests specific genre not represented in watchlist

Always cross-check results against `kinopoisk_ratings.md` to avoid suggesting already watched films.
