# Meetify - Руководство пользователя

## Быстрый старт

### Создание комнаты

1. Откройте приложение в браузере
2. Нажмите "Создать комнату" (или перейдите по прямой ссылке)
3. Разрешите доступ к камере и микрофону
4. Поделитесь ссылкой с участниками

### Присоединение к комнате

1. Получите ссылку от организатора
2. Перейдите по ссылке или введите ID комнаты вручную
3. Разрешите доступ к медиаустройствам
4. Дождитесь подключения других участников

## Управление медиа

### Кнопки управления

| Кнопка | Действие | Горячая клавиша |
|--------|----------|-----------------|
| 🎤 | Включить/выключить микрофон | — |
| 📹 | Включить/выключить камеру | — |
| 🖥️ | Начать/остановить демонстрацию экрана | — |
| ⏺️ | Начать/остановить запись | — |
| ✋ | Поднять/опустить руку | — |
| 📞 | Выйти из комнаты | — |

### Демонстрация экрана

1. Нажмите кнопку 🖥️
2. Выберите вкладку или окно для демонстрации
3. Нажмите "Поделиться"
4. Чтобы остановить — нажмите "Остановить" в плашке браузера или кнопку 🛑

## Чат

### Отправка сообщений

1. Введите текст в поле внизу чата
2. Нажмите Enter или кнопку ➤

### Системные сообщения

В чате отображаются:
- Кто присоединился/вышел
- Кто начал/остановил запись
- Результаты опросов
- Статус поднятой руки

## Виртуальная доска

### Открытие

Нажмите кнопку 🎨 — доска откроется поверх видео.

### Инструменты

- **Карандаш** — рисование (выберите цвет)
- **Ластик** — стирание
- **Очистить** — полная очистка доски

### Особенности

- Все участники видят рисование в реальном времени
- Доска общая для всех в комнате
- При закрытии содержимое не сохраняется

## Опросы

### Создание опроса

1. Нажмите кнопку 📊
2. Введите вопрос
3. Добавьте варианты ответа (минимум 2)
4. Выберите тип: анонимный или публичный
5. Нажмите "Создать опрос"

### Голосование

1. Выберите один из вариантов
2. Результаты обновляются в реальном времени
3. Можно изменить свой выбор

### Закрытие опроса

Только создатель может закрыть опрос кнопкой "Закрыть опрос".

## Реакции

1. Нажмите кнопку 😊
2. Выберите эмодзи
3. Реакция появится анимацией над вашим видео

Доступные реакции: 👍 ❤️ 😂 😮 😢 🎉

## Запись встречи

### Начало записи

1. Нажмите кнопку ⏺️
2. Подтвердите разрешение на уведомления (опционально)
3. Индикатор покажет время записи

### Во время записи

- Записывается видео всех участников (сетка)
- Аудио микшируется
- Запись продолжается даже если вы свернёте вкладку

### Остановка и скачивание

1. Нажмите ⏹️ для остановки
2. В чате появится ссылка для скачивания
3. Или откройте 📹 "Записи" для списка всех записей

## Виртуальный фон

### Настройка

1. Нажмите ⚙️ "Настройки"
2. Выберите "Виртуальный фон"
3. Доступные варианты:
   - Без фона
   - Размытие
   - Сплошной цвет
   - Пресеты: Офис, Природа, Космос, Библиотека
   - Своя картинка

### Примечание

Для фонов с картинками требуется загрузка ML-модели (~10 секунд).

## Настройки

### Качество видео

- **Высокое** (1280×720) — хорошее качество, больше трафика
- **Среднее** (854×480) — баланс качества и трафика
- **Низкое** (640×360) — экономия трафика

### Папка для записей

Укажите путь для автоматического сохранения записей.

## Участники

### Просмотр списка

Нажмите 👥 — откроется боковая панель со списком участников.

### Поднятие руки

Нажмите ✋ — все увидят уведомление и иконку на вашем видео.

## Приглашения

### Поделиться комнатой

1. Нажмите ✉️ в шапке
2. Скопируйте ссылку или поделитесь через:
   - Telegram
   - Email
   - WhatsApp

## Устранение неполадок

### Нет видео/аудио

1. Проверьте разрешения браузера
2. Перезагрузите страницу
3. Проверьте, не используется ли камера другим приложением

### Не подключаются участники

1. Проверьте интернет-соединение
2. Попробуйте VPN (возможны блокировки WebRTC)
3. Проверьте настройки firewall

### Плохое качество связи

1. Переключитесь на более низкое качество видео
2. Отключите камеру, оставьте только аудио
3. Закройте лишние вкладки браузера

### Доска не работает

- Убедитесь, что используете современный браузер
- Попробуйте очистить кэш
